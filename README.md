# dotfiles

## Usage

**Clone the repo**:

```bash
git clone https://github.com/khaleelsyed/dotfiles.git
```

**Install the dot files**:

_**NOTE**: This requires sudo access (installs `stow`)_

```bash
cd dotfiles
bash install.sh
```
